# Tweeter - Twitter Clone
This is a Twitter clone app made with ```.NET Core 5``` and ```Vue.js```.

# DotNet Used Commands

## Build
```
dotnet build
dotnet run
```

## Migrations
```
dotnet ef migrations add "Name of migration"
```

```
dotnet ef database update
```

## Add Packages

```
dotnet add package Microsoft.Asp....
```
